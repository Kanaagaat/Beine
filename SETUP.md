# Quick Setup Guide

## Step 1: Install Dependencies

```bash
npm install
```

## Step 2: Run Development Server

```bash
npm run dev
```

Visit [http://localhost:3000/ru](http://localhost:3000/ru) or [http://localhost:3000/kk](http://localhost:3000/kk)

**That's it!** No database, no environment variables, no additional setup needed.

## Testing

Run tests:
```bash
npm test
```

## Production Deployment

1. Deploy to Vercel, Railway, or your preferred platform
2. No environment variables or database setup required
3. The app is ready to use!

## Editing Content

- **Locations**: Edit `src/data/locations.ts`
- **Gallery**: Edit `src/data/gallery.ts`
- **Pricing**: Edit `src/config/pricing.ts`
- **Translations**: Edit `messages/ru.json` and `messages/kk.json`
